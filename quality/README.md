# Quality

Skills for confirming that work actually works.

| Skill | Purpose |
|-------|---------|
| [`verify`](./verify) | Generate and optionally execute end-to-end verification steps after completing a plan or implementation. |
| [`find-critical-gaps`](./find-critical-gaps) | Pressure-test a freshly grilled plan or just-implemented change for data-integrity, security, failure-mode, and architectural risks. |
